Rotation エンティティ自体

transformation とは別に、エンティティの向き Rotation:[yaw,pitch] を回します。 yaw は 0 が南（+Z）で、正の値で西向きへ。pitch は正の値で下向き。全体が回るので translation の方向ごと変わります。

26.2 でも item_display のアイテムは item:{id:"…",count:1} のままです。1.20.4 以前を選ぶと Count:1b 形式になります。
入力はオイラー角（度・XYZ順）、出力はクォータニオン [x,y,z,w]。 item_display の値はゲーム内でモデルに追加の傾きと縮小をかけるため、プレビューは transformation と Rotation のみを再現しています。
